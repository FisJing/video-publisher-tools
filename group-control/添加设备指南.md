# 添加新设备指南

## 概述

本系统使用 **ADB无线连接** 方式控制手机，不需要在手机上安装任何APP。

但为了支持中文输入，需要安装 **ADBKeyboard** 输入法。

---

## 手机端准备

### 必须设置

| 设置项 | 说明 | 是否必须 |
|--------|------|----------|
| 开发者选项 | 启用高级功能 | ✅ 必须 |
| USB调试 | ADB连接基础 | ✅ 必须 |
| ADBKeyboard | 中文输入 | ⚠️ 可选（推荐安装） |

---

## 第一步：开启开发者选项

### Android 手机通用方法

```
设置 → 关于手机 → 连续点击"版本号" 7次
```

### 不同品牌手机位置

| 品牌 | 路径 |
|------|------|
| 小米/红米 | 设置 → 我的设备 → 全部参数 → MIUI版本 |
| 华为/荣耀 | 设置 → 关于手机 → 版本号 |
| OPPO/vivo | 设置 → 关于手机 → 软件版本号 |
| 三星 | 设置 → 关于手机 → 软件信息 → 编译编号 |
| 原生Android | 设置 → 关于手机 → 版本号 |

### 成功提示

```
"您已处于开发者模式" 或 "无需进行此操作"
```

---

## 第二步：开启USB调试

### 路径

```
设置 → 开发者选项 → 开启"USB调试"
```

### 部分手机额外设置

| 品牌 | 额外设置 |
|------|----------|
| 小米/红米 | 开启"USB调试（安全设置）" |
| 华为/荣耀 | 开启"仅充电模式下允许ADB调试" |
| OPPO/vivo | 需要输入验证码 |

---

## 第三步：获取手机IP地址

### 方法

```
设置 → WLAN → 点击已连接的网络名称 → 查看"IP地址"
```

### 示例

```
IP地址: 192.168.31.168
```

### 注意

- 手机和电脑必须连接**同一WiFi**
- 建议在路由器设置**静态IP**，避免IP变化

---

## 第四步：连接ADB（首次必须有线）

### 4.1 USB连接手机

1. 用USB数据线连接手机和电脑
2. 手机上弹出"允许USB调试"，点击**允许**
3. 勾选"始终允许"

### 4.2 验证连接

打开电脑命令行，运行：

```bash
adb devices
```

应该看到：

```
List of devices attached
XXXXXXXX    device
```

### 4.3 开启无线ADB

```bash
# 开启无线调试端口
adb tcpip 5555

# 连接手机IP
adb connect 手机IP:5555

# 例如
adb connect 192.168.31.168:5555
```

成功提示：

```
connected to 192.168.31.168:5555
```

### 4.4 拔掉USB线

无线连接成功后，可以拔掉USB线。

---

## 第五步：添加到系统配置

### 编辑配置文件

打开 `device-config.json`，添加新设备：

```json
{
  "192.168.31.163:5555": { "alias": "资产-13", "description": "视频号" },
  "192.168.31.167:5555": { "alias": "资产-5", "description": "视频号" },
  "192.168.31.168:5555": { "alias": "新设备名", "description": "视频号" }
}
```

### 配置说明

| 字段 | 说明 |
|------|------|
| IP:端口 | 手机IP地址 + :5555 |
| alias | 设备别名（自定义名称） |
| description | 设备描述 |

### 重启服务器

修改配置后，重启服务器使更改生效。

---

## 第六步：安装ADBKeyboard（推荐）

### 为什么需要？

系统需要通过ADB输入中文描述，默认输入法不支持。
**ADBKeyboard** 是一个专门用于ADB输入的输入法。

### 下载地址

```
GitHub: https://github.com/senzhk/ADBKeyBoard
直接下载: https://github.com/senzhk/ADBKeyBoard/raw/master/ADBKeyboard.apk
```

### 安装方法

#### 方式一：ADB安装（推荐）

```bash
adb install ADBKeyboard.apk
```

#### 方式二：手机安装

1. 把APK传到手机
2. 点击安装

### 启用输入法

安装后，需要手动启用：

```
设置 → 语言和输入法 → 虚拟键盘 → 启用"ADB Keyboard"
```

或者在命令行运行：

```bash
adb shell ime enable com.android.adbkeyboard/.AdbIME
```

### 验证安装

```bash
adb shell ime list -s | grep AdbIME
```

如果显示 `com.android.adbkeyboard/.AdbIME` 则安装成功。

## 常见问题

### Q1: USB调试在哪找不到？

部分手机开发者选项是隐藏的，需要先开启：

```
设置 → 关于手机 → 连续点击版本号7次
```

### Q2: adb devices显示unauthorized？

手机上没有授权USB调试：
1. 查看手机屏幕是否有弹窗
2. 勾选"始终允许"
3. 点击"允许"

### Q3: adb connect失败？

检查：
1. 手机和电脑是否同一WiFi
2. 手机IP是否正确
3. 手机屏幕是否亮着（锁屏可能断开）

### Q4: 输入中文失败？

需要安装ADBKeyboard：

```bash
# 安装
adb install ADBKeyboard.apk

# 启用
adb shell ime enable com.android.adbkeyboard/.AdbIME
```

### Q5: 手机重启后连接不上？

手机重启后ADB会关闭，需要重新连接：

```bash
# USB连接后
adb tcpip 5555
adb connect 手机IP:5555
```

### Q6: IP地址经常变化？

在路由器设置静态IP：
1. 登录路由器管理页面
2. 找到DHCP设置
3. 为手机绑定固定IP

## 快速命令参考

```bash
# 查看已连接设备
adb devices

# 开启无线ADB
adb tcpip 5555

# 连接设备
adb connect 192.168.31.xxx:5555

# 断开连接
adb disconnect 192.168.31.xxx:5555

# 安装ADBKeyboard
adb install ADBKeyboard.apk

# 启用ADBKeyboard
adb shell ime enable com.android.adbkeyboard/.AdbIME

# 列出输入法
adb shell ime list -s

# 查看手机IP
adb shell ip addr show wlan0
```
